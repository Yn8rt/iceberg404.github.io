也就是说最后要变为7z的压缩包，大佬脚本（此脚本可一步到位得到的test.txt以16进制文本的形式粘贴到010然后保存格式为）：

import re

def read_file(filepath):
    with open(filepath) as fp:
        content=fp.read();
    return content

number = read_file('1.txt')
result = []
result.append(re.findall(r'.{2}', number))
result = result[0]

strings =''
for i in result:
    y = bytearray.fromhex(i)
    z = str(y)
    z= re.findall("b'(.*?)'",z)[0]
    strings += z

b= strings.split('0x')

strings=''
for i in b:
    if len(i) ==1:
        i= '0' + i
    strings +=i

with open('test.txt', 'w') as f:
    f.write(strings)

这里用base64和32件套的脚本原因是针对开头部分加密字符串进行解密然后判断，以下是脚本（亲测可行）：

import base64
import re

f = open('C:/Users/Administrator/Desktop/secenc.txt').read().encode('utf-8')

while True:
    if re.match('^[2-7A-Z=]+$', f.decode('utf-8')):
        f = base64.b32decode(f)
    elif re.match('^[0-9a-zA-Z+/=]+$', f.decode('utf-8')):
        f = base64.b64decode(f)
    else:
        print(f.decode('utf-8'))
        break

红包题第一弹

打开得到86个zip文件，直接86个图片，尝试解压几张图片，并用十六进制编辑器查看，发现末尾均有额外数据，基本就是base64。

脚本介绍：按文件名顺序依次将压缩包解压得到图片，并提取图片末尾的字符串后拼接：

import zipfile
path1 = 'C:/Users/Administrator/Desktop/flag/'
path2 = 'C:/Users/Administrator/Desktop/flag/out/'

res = ''
for i in range(1, 87):
    zip = zipfile.ZipFile(path1 + str(i) + '.zip')
    jpg = str(i) + '.jpg'
    zip.extract(jpg, path2)
    zip.close()

    f = open(path2 + jpg, 'rb').read()
    res += f[len(f)-100:len(f)+1].decode('utf-8')
print(res)

关于base64编码的图片base64的开头为iVBOR…此处就是，所以文件头就加上data:image/png;base64,然后base64转图片得到二维码得：flag{gif_is_so_easy}

import zipfile
import string
import binascii

crc = []
zip = zipfile.ZipFile('C:/Users/Administrator/Desktop/flag.zip', 'r')
for i in zip.namelist():
    crc += [zip.getinfo(i).CRC]
crc = crc[1:10]

for i in range(9):
    for j in string.printable:
        c = binascii.crc32(j.encode('utf-8'))
        if c == crc[i]:
            print(j, end = '')
